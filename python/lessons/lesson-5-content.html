Bài 5: Ra Quyết Định (Making Decisions)

Bài học này giới thiệu cách chương trình Python có thể "ra quyết định" dựa trên các điều kiện, giúp chương trình trở nên thông minh và linh hoạt hơn.

Mục tiêu Bài học:

  • Hiểu khái niệm về giá trị Chân lý (Boolean): True (Đúng) và False (Sai).
  • Sử dụng các phép toán so sánh (==, !=, >, <, >=, <=) để tạo ra các biểu thức điều kiện.
  • Học và sử dụng câu lệnh if để thực thi mã dựa trên một điều kiện.
  • Sử dụng câu lệnh else để cung cấp một hành động thay thế khi điều kiện if không đúng.
  • Sử dụng câu lệnh elif (else if) để kiểm tra nhiều điều kiện một cách tuần tự.
  • Sử dụng các phép toán logic (and, or, not) để kết hợp nhiều điều kiện.
  • Hiểu và viết các câu lệnh if lồng nhau (nested if).
  • Áp dụng kiến thức để giải các bài toán yêu cầu ra quyết định.

Giới thiệu:

Trong cuộc sống hàng ngày, chúng ta liên tục phải đưa ra quyết định. Ví dụ: "Nếu trời mưa, tôi sẽ mang ô. Ngược lại, nếu trời nắng, tôi sẽ đội mũ." Máy tính cũng cần khả năng "ra quyết định" tương tự để thực hiện các công việc một cách thông minh và linh hoạt.

Trong Python, chúng ta sử dụng các câu lệnh điều kiện (conditional statements) để cho phép chương trình lựa chọn thực hiện những khối lệnh khác nhau dựa trên việc một điều kiện nào đó là Đúng (True) hay Sai (False). Bài học này sẽ hướng dẫn cách sử dụng các câu lệnh này.

1. Giá trị Chân lý (Boolean Values) và Biểu thức Điều kiện

a. Giá trị Chân lý: True và False

Như đã giới thiệu ở Bài 2, kiểu dữ liệu Boolean chỉ có hai giá trị:

  • True: Đại diện cho sự thật, điều đúng đắn.
  • False: Đại diện cho sự giả dối, điều sai trái.

(Lưu ý: True và False trong Python phải viết hoa chữ cái đầu).

b. Phép toán So sánh (Comparison Operators)

Các phép toán so sánh được sử dụng để so sánh hai giá trị và kết quả của phép so sánh luôn là một giá trị Boolean (True hoặc False).

Ví dụ sử dụng phép toán so sánh:

x = 10
y = 20

ket_qua_1 = (x == 10)       # x có bằng 10 không?
ket_qua_2 = (y < x)         # y có nhỏ hơn x không?
ket_qua_3 = (x != y)        # x có khác y không?

print(f"x == 10 là: {ket_qua_1}")   # Kết quả: True
print(f"y < x là: {ket_qua_2}")     # Kết quả: False
print(f"x != y là: {ket_qua_3}")   # Kết quả: True

Những biểu thức như x == 10 hay y < x được gọi là biểu thức điều kiện vì chúng trả về một giá trị True hoặc False dùng để kiểm soát luồng của chương trình.

2. Câu lệnh if (The if Statement)

Câu lệnh if cho phép thực thi một khối lệnh nếu một điều kiện cụ thể là True.

Cú pháp:

if dieu_kien:
    # Khối lệnh sẽ được thực thi nếu dieu_kien là True
    # Các lệnh trong khối này phải được thụt vào đầu dòng (indentation)
    lenh_1
    lenh_2
    ...
# Các lệnh bên ngoài khối if (không thụt vào) sẽ luôn được thực thi
lenh_tiep_theo
  • dieu_kien: Là một biểu thức trả về True hoặc False.
  • Dấu hai chấm (:) là bắt buộc sau dieu_kien.
  • Thụt lề (Indentation): Các dòng lệnh thuộc khối if phải được thụt vào so với dòng if (thường là 4 dấu cách hoặc một phím Tab). Đây là cách Python xác định khối lệnh nào thuộc về if.

Luồng hoạt động:

  1. Python kiểm tra dieu_kien.
  2. Nếu dieu_kien là True, các lệnh bên trong khối if (đã thụt lề) sẽ được thực thi.
  3. Nếu dieu_kien là False, các lệnh bên trong khối if sẽ bị bỏ qua, và chương trình sẽ tiếp tục với lenh_tiep_theo (nếu có).

Ví dụ 1: Kiểm tra tuổi để xem phim

tuoi = int(input("Nhập tuổi của bạn: "))

if tuoi >= 18:
    print("Bạn đủ tuổi để xem phim này.")
    print("Chúc bạn xem phim vui vẻ!")

print("Cảm ơn bạn đã sử dụng dịch vụ.") # Lệnh này luôn được thực thi

3. Câu lệnh else (The else Statement)

Câu lệnh else được sử dụng cùng với if để cung cấp một khối lệnh thay thế sẽ được thực thi nếu điều kiện của if là False.

Cú pháp:

if dieu_kien:
    # Khối lệnh thực thi nếu dieu_kien là True
    lenh_A1
    lenh_A2
else:
    # Khối lệnh thực thi nếu dieu_kien là False
    lenh_B1
    lenh_B2
# Các lệnh tiếp theo

Luồng hoạt động:

  1. Python kiểm tra dieu_kien của if.
  2. Nếu dieu_kien là True, khối lệnh của if được thực thi, và khối lệnh của else bị bỏ qua.
  3. Nếu dieu_kien là False, khối lệnh của if bị bỏ qua, và khối lệnh của else được thực thi.

Ví dụ 2: Kiểm tra số chẵn lẻ

so = int(input("Nhập một số nguyên: "))

if so % 2 == 0:  # Nếu số dư khi chia cho 2 là 0
    print(f"Số {so} là số chẵn.")
else:
    print(f"Số {so} là số lẻ.")

4. Câu lệnh elif (The elif Statement)

elif là viết tắt của "else if". Nó cho phép kiểm tra nhiều điều kiện một cách tuần tự. Nếu điều kiện if đầu tiên sai, Python sẽ kiểm tra điều kiện của elif tiếp theo, và cứ thế.

Cú pháp:

if dieu_kien_1:
    # Khối lệnh cho dieu_kien_1
    ...
elif dieu_kien_2:
    # Khối lệnh cho dieu_kien_2 (chỉ thực thi nếu dieu_kien_1 False và dieu_kien_2 True)
    ...
elif dieu_kien_3:
    # Khối lệnh cho dieu_kien_3 (chỉ thực thi nếu dieu_kien_1, dieu_kien_2 False và dieu_kien_3 True)
    ...
else: # Tùy chọn, có thể có hoặc không
    # Khối lệnh thực thi nếu tất cả các điều kiện trên đều False
    ...

Ví dụ 3: Xếp loại học lực dựa trên điểm số

diem_trung_binh = float(input("Nhập điểm trung bình của bạn: "))

if diem_trung_binh >= 8.0:
    print("Xếp loại: Giỏi")
elif diem_trung_binh >= 6.5:
    print("Xếp loại: Khá")
elif diem_trung_binh >= 5.0:
    print("Xếp loại: Trung bình")
else:
    print("Xếp loại: Yếu")

5. Phép toán Logic (and, or, not)

Các phép toán logic được sử dụng để kết hợp nhiều biểu thức điều kiện lại với nhau.

a. and (Và)

Trả về True nếu cả hai biểu thức đều True.

Ví dụ 4: Kiểm tra điều kiện đi học

khoe_manh = True
lam_bai_tap = False

if khoe_manh and lam_bai_tap:
    print("Bạn đủ điều kiện đi học.")
else:
    print("Bạn chưa đủ điều kiện đi học.")

b. or (Hoặc)

Trả về True nếu ít nhất một trong hai biểu thức là True.

Ví dụ 5: Kiểm tra điều kiện giảm giá

la_cuoi_tuan = False
co_the_thanh_vien = True

if la_cuoi_tuan or co_the_thanh_vien:
    print("Bạn được giảm giá!")
else:
    print("Bạn không được giảm giá.")

c. not (Phủ định)

Đảo ngược giá trị Boolean của một biểu thức.

Ví dụ 6: Kiểm tra nếu không phải cuối tuần

la_cuoi_tuan = False

if not la_cuoi_tuan:
    print("Hôm nay là ngày thường, đi làm thôi!")
else:
    print("Hôm nay là cuối tuần, nghỉ ngơi nào!")

6. Câu lệnh if lồng nhau (Nested if Statements)

Một câu lệnh if có thể chứa một câu lệnh if khác bên trong nó.

Ví dụ 7: Kiểm tra đăng nhập và quyền truy cập

ten_dang_nhap = input("Nhập tên đăng nhập: ")
mat_khau = input("Nhập mật khẩu: ")

if ten_dang_nhap == "admin":
    if mat_khau == "12345":
        print("Đăng nhập thành công với quyền quản trị!")
    else:
        print("Sai mật khẩu.")
elif ten_dang_nhap == "user":
    if mat_khau == "user123":
        print("Đăng nhập thành công với quyền người dùng.")
    else:
        print("Sai mật khẩu.")
else:
    print("Tên đăng nhập không tồn tại.")

7. Bài tập

  1. Kiểm tra số dương, số âm, hay số không: Nhập số nguyên, in ra là dương, âm hay không.
  2. Giải phương trình bậc nhất $ax + b = 0$: Nhập a, b. Biện luận và giải (nghiệm $x = -b/a$).
  3. Tính tiền điện: Nhập số kWh. Tính tiền theo bậc thang.
  4. Kiểm tra năm nhuận: Nhập năm. Kiểm tra có phải năm nhuận không.
  5. Đoán số (đơn giản): Đặt số bí mật. Người dùng đoán. So sánh và thông báo.
  6. Phân loại tam giác: Nhập 3 cạnh a, b, c. Phân loại đều, cân, vuông ($a^2 + b^2 = c^2$), thường.
  7. Kiểm tra điều kiện tham gia câu lạc bộ: Dựa trên tuổi và điểm năng khiếu.

8. Tóm tắt Bài 5 và Từ khóa

Bài học này đã giới thiệu cách để chương trình Python đưa ra quyết định dựa trên các điều kiện.
